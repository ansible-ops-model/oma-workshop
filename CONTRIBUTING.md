# Contributing to the OMA Workshop Tool

## How to propose a new topic
Open an issue using the Topic Proposal template.

## How to improve guidance text
Edit the `GUIDE` constant in `index.html` and open a pull request.
Guidance keys must match topic names exactly (case-sensitive).

## How to report a bug
Open an issue using the Bug Report template.

## Pull request process
1. Fork the repository
2. Make your changes
3. Open a PR with a clear description of what changed and why
4. One maintainer review required before merge

## Model content
This tool is built on the model at https://ansible-ops-model.gitlab.io/
Changes to model content (what topics exist, how they're described) should
be proposed there first.
