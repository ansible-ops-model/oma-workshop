How to use the OMA Workshop Tool

This tool helps you benchmark your organisation against the Operational Model for Ansible and prioritise what to work on next.

Navigating the tree

Use View to set how many levels are visible. Click any node's ▾ indicator to expand or collapse that branch individually. Hover over leaf nodes to see a description and discussion prompts.

Annotating topics

Select a Mode, then click any node to apply it. Modes:

  • Navigate — click nodes to expand/collapse (default)
  • ✓ Have this — mark topics your organisation already has in place
  • ◑ Planned — mark topics on your roadmap
  • MVP / 2.0 / Advanced — reclassify a topic's maturity level
  • ✕ Clear — remove an annotation from a node

Keyboard shortcuts

  • Esc — return to Navigate mode
  • 1 2 3 — switch to Pillars / Groups / All topics view

Saving your work

The tool auto-saves to your browser on every change. Use ↓ Save snapshot to download a named JSON file you can share or reload later. Drag a snapshot file onto the page to restore a session.

Adding custom topics

Use + Add topic to add topics specific to your organisation. Custom topics are marked with a dashed border and ★ in exports.
